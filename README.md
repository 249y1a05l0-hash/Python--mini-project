# Number Guessing Game (Python)

## Description
This is a console-based Python game where the user tries to guess a randomly generated number. 
The program provides feedback if the guess is too high or too low until the correct number is guessed.

## Concepts Used
- Python basics
- Loops (while)
- Conditional statements
- User input
- Random module
- Exception handling

## How to Run
1. Make sure Python is installed.
2. Run the program using:
   python python.py
